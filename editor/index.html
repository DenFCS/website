    One-time setup: GitHub token

    To save your edits, the editor needs a GitHub Personal Access Token. This is stored in your browser only — never sent anywhere else.

    1. Open github.com/settings/tokens
    2. Click Generate new token (classic)
    3. Check the repo scope, set expiration, generate
    4. Copy the token (ghp_…) and paste it below
